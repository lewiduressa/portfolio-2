Thread-Safe Bounded Buffer

View Code

I developed a thread-safe bounded buffer in C using the `pthread.h` library, designed for concurrent multi-threaded environments. The buffer is implemented with a circular queue and utilizes mutex locks and condition variables to ensure synchronization. The API includes four essential methods: initialize, push, pop, and destroy (to clean up resources). The use of conditional variables ensures that threads wait when the buffer is full or empty, preventing data races. The project required low-level synchronization primitives.
